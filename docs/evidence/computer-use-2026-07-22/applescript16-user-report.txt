First of all, thanks for the rapid pace of development. I’ve been testing Osaurus extensively on an M1 MacBook Air (16 GB RAM), and 0.22.7 is a noticeable improvement.

Improvements

Computer Use now reliably opens applications (e.g. TextEdit) and correctly reports success instead of immediately failing. This is a significant improvement over previous releases.
General responsiveness also feels better.

Remaining issues

Computer Use retries successful edits

Prompt:

Change the text in the file from “Hello from OracHQ” to “Hello again”.

Expected:
The text is replaced once and the task completes.

Actual:
The agent successfully types “Hello again” but appears not to recognise that it has succeeded. It retries the action multiple times, resulting in:

Hello againHello againHello againHello again

before eventually reporting Failed: Computer use.

It looks like the success verification is failing and causing repeated execution of an already-completed action.

⸻

Unnecessary save attempt

Another test:

Prompt:

Change the text in the file from “Hello World” to “Hello from OracHQ”

The edit was performed correctly.

However, after completing the requested edit, the agent automatically attempted to Save the document, even though I hadn’t requested that. The save workflow then failed, causing the overall task to be marked as failed.

It would be better if the task terminated successfully immediately after completing the requested edit unless saving was explicitly requested.

⸻

Overall, I think 0.22.7 is a big step forward. The remaining issues now seem to be around task completion and success detection rather than basic Computer Use functionality.

Thanks again for all the work!
