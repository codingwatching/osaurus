# Acme Knowledge Base

This is placeholder reference material for the fictional company **Acme Corp**,
used to test Osaurus Knowledge collections end to end.

## What's in here

- **Policies** — customer-facing rules such as refunds and support hours.
- **SOPs** — internal step-by-step procedures like employee onboarding.
- **Standards** — engineering conventions such as the Swift style guide.
- **Runbooks** — operational how-to guides such as the deploy runbook.
- **Templates** — reusable boilerplate such as canned email replies.

## How to use it

Ask a granted agent a question that only these documents can answer. The agent
should search the library, read the relevant section, and cite it — without you
having to point at a specific file.
